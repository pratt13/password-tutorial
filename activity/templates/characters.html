Character set

The number of combinations a hacker must try to get your password is given by the permutation with replacement formula:
Number of possible passwords = (Size of the character set)password length

So, for a password of length 4, if the password is only allowed to be the lowercase alphabet, the number of combinations is
Number of possible passwords: (26)6= 456976

If uppercase and numbers are allowed
Number of possible passwords: (26 + 26 + 10)6= 14776336
This is why passwords suggest using upper/lower/numeric and special characters to increase complexity.
