Password hacking exercise

Welcome to a simple exercise that illustrates what a good password is, and how vulnerable bad ones are.


There are two tasks, the first is to gain access to a url which have poor password protection, the second is to create a function that generates lots of strong passwords.
  • Task One
  • Task Two
Both tasks can be handled in an number of different ways, but the skills they cover include are given below.

Task One

Minimum skills used:
  • For loops
  • List creation
  • String concatenation, i.e. joining strings (numbers/letters/words) together.
Advanced skills
  • List comprehension
  • Using existing python libraries such as itertools
  • Use of generators

Task Two

Minimum skills used:
  • Use of random library
  • String concatenation, i.e. joining strings (numbers/letters/words) together.
