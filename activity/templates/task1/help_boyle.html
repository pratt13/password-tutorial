Task 1: Help Boyle

The aim is to write a function that returns a list of all possible 3 pin combinations. So the first thing we could do is to use a for loop per character. Then for each guess you could add it to a list using append. 
    
      my_list = []
      for letter2 in ["a", "b", "c"]:
        for letter1 in ["1", "2", "3"]:
          my_list.append(letter1+letter2)
      print(my_list)
