Task 2

Task 2 is about writing a function in python that produces a strong password. In addition, though this will want to produce lots of random passwords, not just one really strong one. By producing loads of passwords, then we can test they are random, and not just spitting out the same password over and over again (or in a repeated pattern).

Task one is split into several parts with increasing complexity. Level 1 do not technically need to be automated, you could write one manually via the links and processing it in the browser. However, level 4 and level 5 will need to be automated.
